HAL started as a Claude wrapper. But the architecture — spawn a CLI subprocess, stream structured JSON output — generalizes to any AI agent that speaks the same protocol.

Today HAL supports:

- **Claude** (Anthropic) — the default
- **Codex** (OpenAI)
- **GitHub Copilot**
- More coming

---

## Switching engines

From Telegram:

```
/engine codex
```

HAL stores the engine choice in your session. Every message from that point forwards uses Codex. Switch back anytime:

```
/engine claude
```

---

## Setting a default engine per project

In `hal.config.yaml`:

```yaml
projects:
  my-app:
    cwd: "/path/to/my-app"
    engine: codex
```
