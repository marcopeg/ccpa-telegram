## Data Directory: `.ccpa/` → `.telegrapp/`

All internal application directories move under `.telegrapp/`:

```
.telegrapp/
  config.json          ← was ccpa.config.json (in cwd)
  config.local.json    ← was ccpa.config.local.json (in cwd)
  users/{userId}/      ← was .ccpa/users/{userId}/
  hooks/               ← was .ccpa/hooks/
  commands/            ← was .ccpa/commands/
  logs/                ← was .ccpa/logs/
```

The config is no longer a top-level file in the working directory — it lives inside `.telegrapp/` alongside user data, hooks, and commands.

The `init` command should create `.telegrapp/config.json` (not `ccpa.config.json`).
