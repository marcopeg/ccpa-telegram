name: omt
description: forces the model to ask more during review
---

Answer to these questions:

1. Is there anything that we still need to discuss before closing the review?
2. What is missing in our discussion?

Then use the answers to produce more questions to the user with the goal of improve the task's refinement.

If you are sure there is nothing more to ask and refine, say "I'm ok" and propose to move to planning.
