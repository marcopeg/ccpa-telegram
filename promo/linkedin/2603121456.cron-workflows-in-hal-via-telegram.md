HAL applies agentic management to local folders via Telegram, so you can run coding workflows from a chat when you leave your computer.

In `v1.3.0`, we expanded that with stronger cron automation:
- project-level cron jobs
- `scheduleStarts` and `scheduleEnds`
- relative schedules

Result: you can automate recurring prompts with clear execution windows across multiple local projects, with less manual babysitting.

Cron docs and examples:
https://github.com/marcopeg/hal/blob/main/docs/crons/README.md
